1️⃣ What is the difference between var, let, and const? Answer: var, let and const are used to declare variables in JavaScript, but they act differently.

. var is the old way to declare variables. It can be redeclared and updated.
. let is the modern way to declare variables. It has block scope, and it can be updated but cannot be redeclared.
. const is also a modern keyword to declare variables and block scoped. It's value cannot be changed after it is   declared and it can not be redeclared.
2️⃣ What is the spread operator (...)? Answer: The spread operator (...) is used to spread array elements or copy elements from an array or object. It helps to combine arrays, copy arrays, or pass multiple values easily.

3️⃣ What is the difference between map(), filter(), and forEach()? Answer: These three methods are used to loop through arrays, but they work different ways. map(): Transforms each element and return a new array. filter():Keeps elements that match a condition and return in a new array. forEach():Just loops through the array and do function but does not return a new array.

4️⃣ What is an arrow function? Answer: An arrow function is a shorter way to write functions in JavaScript. Arrow functions make the code shorter and cleaner.

5️⃣ What are template literals? Answer: Template literals are a modern way to write strings in JavaScript using backticks ( ) instead of quotes. They allow to insert variables directly inside strings. They also support multi-line strings.
